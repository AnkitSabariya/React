React Productivity Setup (Vite + Tailwind)

Fast setup for React development using Vite + Tailwind + VSCode productivity tools.

--- Aliases ---

Mac (zsh/bash)
1. Open Terminal
   Cmd + Space → Terminal
2. Open config file
   nano ~/.zshrc   # for zsh
   nano ~/.bashrc  # for bash
3. Add aliases
  # Dev server start karne ke liye
  alias d="npm run d"

# Production build ke liye
  alias b="npm run b"

 # Dependencies install ke liye
 alias i="npm install"

 # Naya package add karne ke liye
 alias a="npm install"

 # Update sab packages
 alias u="npm update"

# Start server (alternative)
 lias s="npm start"

4. Save & exit
   Ctrl + O → Enter → Ctrl + X
5. Reload config
   source ~/.zshrc

Windows (PowerShell)
1. Open PowerShell
   Start → PowerShell
2. Open profile file
   notepad $PROFILE
3. Add aliases
   Set-Alias rvite "npm create vite@latest"
   Set-Alias dev "npm run dev"
   Set-Alias build "npm run build"
4. Save & restart PowerShell

--- Terminal Shortcuts ---
Action                Mac                 Windows
New terminal          Cmd + Shift + `     Ctrl + Shift + `
Switch terminal tab   Cmd + ↑ / ↓         Ctrl + ↑ / ↓
Kill terminal         Cmd + Shift + W     Ctrl + Shift + W
Clear terminal        Cmd + K            Ctrl + L
